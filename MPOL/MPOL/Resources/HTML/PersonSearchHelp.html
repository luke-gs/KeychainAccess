General Conditions

A person can be searched by name or ID number.

  • Name search is NOT case sensitive and must be in alpha format unless specified otherwise.
  • ID Number search must be in numeric format only.

Search Order

(Last Name*) (,) (Given Name) (Middle Name/s) (DOB/Age Range) (Gender)

Only the Last Name is required

Names

Given names can either be initials or partial names.

eg. (Doe John), (Doe J)

Use a comma to seperate Names that contain spaces.

eg. (de Jaager, Jesse), (Le Gall, Léa)

Names can contain apostrophes or hyphens.

eg. (O'Connor James), (Smith Anne-Marie)

Date of Birth and Age Ranges

Enter these values after the name for a specific age.

eg. (Doe John 1985), (Doe John 06/1985), (Doe John 15/06/1985), (Doe John 25-30)

Enter a single value to return +/-5 years.

eg. (Doe John 30) will return all matching results between the ages of 25-35.

Genders

Enter either M or F last.

eg. (Doe John R 31 M), (Smith Anne-Marie F)

Wildcard Search

Supported for ID numbers only.

  • Use a (?) to represent one unknown number.

  eg. 100748? will return 1007482

  • Use a (*) to represent multiple unknown numbers.

  eg. 1007* will return 10076201 and 1007423
