General Conditions

Vehicles can be searched by registration, engine number or VIN.

Registration

  • Maximum of 9 alpha numeric characters.

eg. ABC123

Engine Number

  • Maximum of 20 alpha numeric characters.

eg. CM80452

VIN

  • Minimum of 8 alpha numeric characters.

eg. 1HGCM80452A008175

Wildcard Search

Supported for all vehicle searches.

  • Use a (?) to represent one unknown character.

eg. AB?123 will return ABC123 and AB0123

  • Use a (*) to represent multiple unknown characters.

eg. ABC12* will return ABC1234 and ABC12004
